# 负面状态系统使用说明

## 系统概述

这是一个完整的负面状态系统，实现了子弹击中敌人时应用中毒效果的功能。中毒效果会让敌人每0.2秒受到1点伤害，持续5秒。

## 核心组件

### 1. BuffSystem.cs
- **Buff基类**: 所有状态效果的基础类
- **BuffManager**: 管理对象身上的所有buff效果

### 2. PoisonBuff.cs
- 中毒负面状态的具体实现
- 每0.2秒造成1点伤害，持续5秒
- 包含视觉效果（颜色变化、闪烁等）

### 3. Health.cs
- 血量管理系统
- 处理伤害、治疗、死亡
- 提供事件回调

### 4. Bullet.cs
- 子弹系统
- 检测碰撞并应用负面状态
- 支持直接伤害 + 中毒效果

### 5. Weapon.cs
- 武器系统，发射带有中毒效果的子弹
- 可配置射速、伤害、中毒参数

### 6. Enemy.cs
- 示例敌人脚本
- 自动设置所需组件
- 简单AI和视觉效果

## 在Unity中设置

### 步骤1：创建玩家武器
1. 创建一个空对象作为玩家
2. 添加`Weapon.cs`脚本
3. 配置武器参数：
   - Fire Rate: 射速（每秒发射次数）
   - Bullet Speed: 子弹速度
   - Bullet Damage: 子弹直接伤害
   - Poison Damage: 中毒伤害
   - Poison Interval: 中毒间隔（0.2秒）
   - Poison Duration: 中毒持续时间（5秒）

### 步骤2：创建敌人
1. 创建一个对象（Cube或其他形状）
2. 添加`Enemy.cs`脚本
3. 设置标签为"Enemy"
4. 脚本会自动添加所需的Health和BuffManager组件

### 步骤3：设置场景
1. 确保敌人对象有Collider组件
2. 为玩家设置"Player"标签（可选，用于敌人AI）
3. 按空格键发射子弹

## 使用方法

### 基本使用
```csharp
// 手动给敌人应用中毒效果
BuffManager buffManager = enemy.GetComponent<BuffManager>();
PoisonBuff poison = new PoisonBuff(1f, 0.2f, 5f); // 伤害, 间隔, 持续时间
buffManager.AddBuff(poison);
```

### 自定义中毒效果
```csharp
// 创建更强的中毒效果
PoisonBuff strongPoison = new PoisonBuff(
    damage: 2f,      // 每次2点伤害
    interval: 0.1f,  // 每0.1秒触发
    duration: 10f    // 持续10秒
);
```

### 创建其他负面状态
```csharp
public class SlowBuff : Buff
{
    private float slowFactor = 0.5f;
    
    public SlowBuff(float duration) : base("slow", "减速", duration) { }
    
    protected override void OnApply()
    {
        // 减速逻辑
    }
    
    // 实现其他抽象方法...
}
```

## 配置参数

### 武器配置
- **Fire Rate**: 1.0 = 每秒1发
- **Bullet Speed**: 子弹飞行速度
- **Bullet Damage**: 击中时的直接伤害
- **Use Poison Bullets**: 是否启用中毒效果

### 中毒配置
- **Poison Damage**: 每次中毒伤害 (默认: 1)
- **Poison Interval**: 伤害间隔 (默认: 0.2秒)
- **Poison Duration**: 持续时间 (默认: 5秒)

### 敌人配置
- **Max Health**: 最大血量
- **Move Speed**: 移动速度
- **Detection Range**: 检测玩家的范围

## 调试功能

### 右键菜单选项
- **武器**: "测试开火" - 立即发射一发子弹
- **敌人**: "测试受伤" - 造成10点伤害
- **敌人**: "应用中毒效果" - 直接应用中毒状态
- **血量**: "测试伤害10点" - 造成10点伤害
- **血量**: "恢复满血" - 恢复到满血状态

### 控制台输出
系统会输出详细的调试信息：
- 子弹击中事件
- 伤害数值
- 中毒效果应用和触发
- 敌人死亡事件

## 扩展性

### 添加新的负面状态
1. 继承`Buff`基类
2. 实现抽象方法
3. 在合适的地方应用新buff

### 添加视觉效果
1. 在`PoisonBuff.ShowPoisonEffect()`中添加粒子效果
2. 在`PoisonBuff.ShowDamageEffect()`中添加数字飘字
3. 使用Unity的粒子系统或UI系统

### 添加音效
在相应的方法中使用AudioSource播放音效：
- 子弹击中音效
- 中毒效果音效
- 敌人死亡音效

## 注意事项

1. 确保敌人对象有"Enemy"标签
2. Bullet需要Collider且设置为Trigger
3. 相同类型的buff会刷新持续时间而不是叠加
4. 敌人死亡时会自动清除所有buff效果
5. 系统支持多种buff同时存在

## 性能优化建议

1. 使用对象池管理子弹
2. 限制同时存在的buff数量
3. 定期清理过期的效果对象
4. 对于大量敌人，考虑使用更高效的更新机制
